## Agregar Persistencia de datos con API

### Enunciado de la actividad

A partir del ejercicio realizado en la tarea anterior, incluir que exista persistencia remota de las categorías y productos, haciendo uso de la API con su documentación (colgada esta semana). Utilizar los métodos GET, POST, PUT y DELETE (omitir el endpoint de alérgenos). El usuario_id de cada uno es los últimos cuatro números de su DNI/NIE.
